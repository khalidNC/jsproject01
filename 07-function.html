What is function?

Function let us reuse the code. Let's create a functon. Write 'function' and a space inside the script tag, it creates a new function.

Syntax: function name_of_function() { code insdie }

Example function:

function function1() {
  console.log("hello");
  console.log(2 + 2);
}

How to use the function?

The function is created. Now call the the function by name with brackets and semicolon outside of the function.

Syntax: function();

Example call a function:

function function1() {
  console.log("hello");
  console.log(2 + 2);
}

function1();

This will show the 'hello' word and the calculation '4' in the console.
